## Overview

### The challenge

Users should be able to:

- 電腦版及手機版畫面。
- 在 input 欄位內填入有效網址，點擊按鈕"Shorten it!"，將產生一組縮短網址。
- 點擊按鈕"Copy"，將複製縮短網址至剪貼簿。
- 當 input 欄位為空欄位為空，或是輸入網址有誤，點擊按鈕"Shorten it!"，將顯示錯誤訊息。
